# Empezar

En esta guía de introducción, le mostraremos cómo crear su propia biblioteca
de componentes con Raddix. Aprenderá a:

- Instalar un hook de Raddix
- Crear un componente
- Agregarle estilos

## Antes de empezar

Le recomendamos que tenga instaladas las siguientes
herramientas antes de comenzar:

- Node.js y gestor de paquetes como npm, yarn o pnpm
- React.js para usar Raddix en su aplicación.

Teniendo listo todo lo anterior, en la siguiente sección
empezaremos con la instalación.
