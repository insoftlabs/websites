# Creando un componente

En este tutorial rápido, instalaremos y contruiremos el componente Switch.

## Instalar el hook

Comenzaremos instalando el hook useSwitch.

```bash
pnpm add @raddix/switch
```

## Construyendo el Switch

### Importar el hook
Una vez instalado procederemos a importar useSwitch en nuestro
archivo <code>switch.jsx</code>

```jsx
// switch.jsx
import { useSwitch } from '@raddix/switch';
```

### Implementar el hook

useSwitch devuelve <code>switchProps</code> para que se distribuyan en su elemento.
Obtenga mas información sobre los props que devuelve useSwitch.

```jsx
// switch.jsx
import { useSwitch } from '@raddix/switch';

const Switch = props => {
  const { switchProps } = useSwitch(props);

  return <button {...switchProps}></button>;
};
```

### Agregar los estilos

En este caso usaremos el preprocesador sass para aplicar los estilos. Recuerde
que puede usar estilos en linea, CSS-in-JS, etc.

```scss
/* styles.scss */
.switch {
  all: unset;
  display: block;
  width: 42px;
  height: 22px;
  background-color: rgba(15, 23, 42, 0.6);
  border-radius: 99px;
  position: relative;
  padding: 3px;
  -webkit-tap-highlight-color: black;
  cursor: pointer;
  transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
  transition-duration: 0.2s;
  transform: scale(1.3);

  &::before {
    content: '';
    display: block;
    width: 22px;
    height: 22px;
    background-color: #fff;
    border-radius: 99px;
    transition: transform 100ms;
    transform: translateX(2px);
    will-change: transform;
    cursor: pointer;
  }

  &[data-state='checked'] {
    background-color: var(--color-primary);
  }

  &[data-state='checked']::before {
    transform: translateX(19px);
  }
}
```

Y por último agregamos los estilos a nuestro componente

```jsx
// switch.jsx
import { useSwitch } from '@raddix/switch';
import './styles.scss';

const Switch = props => {
  const { switchProps } = useSwitch(props);

  return <button {...switchProps} className='switch'></button>;
};
```
