# Creating a component

Once @raddix/switch is installed we will proceed to create our component.

## Import the hook

Let's start by importing useSwitch into our <code>switch.jsx</code> file.

```jsx
// switch.jsx
import { useSwitch } from '@raddix/switch';
```

## Implement the hook

useSwitch returns <code>switchProps</code> props to be spread on your element.
Learn more about the props returned by useSwitch.

```jsx
// switch.jsx
import { useSwitch } from '@raddix/switch';

const Switch = props => {
  const { switchProps } = useSwitch(props);

  return <button {...switchProps}></button>;
};
```

## Add the styles

In this case we will use the sass preprocessor to apply the styles.
Remember that you can use inline styles, CSS-in-JS, etc.

```scss
/* styles.scss */
.switch {
  all: unset;
  display: block;
  width: 42px;
  height: 22px;
  background-color: rgba(15, 23, 42, 0.6);
  border-radius: 99px;
  position: relative;
  padding: 3px;
  -webkit-tap-highlight-color: black;
  cursor: pointer;
  transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
  transition-duration: 0.2s;
  transform: scale(1.3);

  &::before {
    content: '';
    display: block;
    width: 22px;
    height: 22px;
    background-color: #fff;
    border-radius: 99px;
    transition: transform 100ms;
    transform: translateX(2px);
    will-change: transform;
    cursor: pointer;
  }

  &[data-state='checked'] {
    background-color: var(--color-primary);
  }

  &[data-state='checked']::before {
    transform: translateX(19px);
  }
}
```

And finally we add the styles to your component

```jsx
// switch.jsx
import { useSwitch } from '@raddix/switch';
import './styles.scss';

const Switch = props => {
  const { switchProps } = useSwitch(props);

  return <button {...switchProps} className='switch'></button>;
};
```
