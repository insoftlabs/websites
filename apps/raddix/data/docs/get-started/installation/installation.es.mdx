# Instalación

Cada hook de Raddix se instala individualmente para que pueda administrar sus
versiones de manera más granular o adoptarlos de forma incremental.
Se puede instalar mediante un administrador de paquetes como npm, yarn o pnpm.

## Instalando useSwitch

Para el ejemplo de esta guia crearemos un componente switch y lo haremos
con el hook useSwitch, asi que procederemos a instalarlo.

<br />
En este caso estamos usando pnpm pero usted puede instalarlo con el administrador
de paquetes de su elección.

```bash
pnpm add @raddix/switch
```
