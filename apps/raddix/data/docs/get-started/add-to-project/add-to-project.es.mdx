# Agregue a su proyecto

Raddix solo se puede usar en proyectos de React, puede usarlo en sus componentes
de su applicación web o adoptarlos de forma incremental en su sistema de diseño.

## Instalación

Puede instalar Raddix usando un administrador de paquetes como npm, yarn o pnpm.

<br />
Cada hook de Raddix se instala individualmente para que pueda administrar sus
versiones de forma individual. Esto le permite instalar solo los hooks que
necesite.

<br />

Por ejemplo, si desea instalar el hook para un componente switch mediante pnpm.
```bash
pnpm add @raddix/switch
```
