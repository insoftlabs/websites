# useSwitch

@raddix/switch is un hook that provides accessibility and behavior for a switch component.

<Demo>
  <Switch />
</Demo>

## Features

- Full support for browser features.
- Full keyboard navigation.
- Labeling support for screen readers.
- Can be controlled or uncontrolled.
- Fully-typed API

## Installation

Install the primitive from your command line.

```bash
yarn add @raddix/switch
```

## Example Basic

This example uses an HTML button tag to build a switch, we just import
the useSwitch hook which will return switchProps and spreading it in our
button tag.

```tsx
// switch.tsx
import { useSwitch, UseSwitchProps } from '@raddix/switch';
import './styles.css';

const Switch = (props: UseSwitchProps) => {
  const { switchProps } = useSwitch(props);

  return <button {...switchProps} className='switch'></button>;
};
```

## Usage

### Default Checked

Switches are not selected by default. To be checked by default
using <code>defaultChecked</code> prop

```tsx
// switch.tsx
import Switch from './switch';

<Switch defaultChecked />;
```

<Demo>
  <Switch defaultChecked />
</Demo>

### Read only

The <code>readOnly</code> prop makes the selection immutable.

```tsx
// switch.tsx
import Switch from './switch';

<Switch defaultChecked readOnly />;
```

<Demo>
  <Switch defaultChecked readOnly />
</Demo>

### Label Form

Switch supports the integration with HTML tag Label

<Demo>
  <label htmlFor='dev' style={{ paddingRight: 20 }} id='dev-3'>
    Living Room Lights
  </label>
  <Switch defaultChecked id='dev' aria-labelledby='dev-3' />
</Demo>
