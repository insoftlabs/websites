# useCountDown

@raddix/useCountDown is a custom hook that does the job of handling a countdown timer for you.

You can render its current value into the DOM as it refreshes, and manipulate its behavior as you wish (like specifying an _onFinished_ callback).

## Installation

Install the custom hook from your command line.

```bash
yarn add @raddix/useCountDown
```

### Return Value

It returns an array that can be destructured in two variables: _count_ and _actions_. These are reference names, as you can name them as preference.
