# useCounter

`useCounter` es un hook para manejar un contador con límites mínimo y máximo.

## Instalación

```sh
yarn add @raddix/use-counter
```

## API

### Valor de retorno

El hook `useCounter` devuelve un array con dos elementos:

<ApiTable
  head={{ name: 'Index' }}
  data={[
    {
      name: '0 (counter)',
      type: 'number',
      description: 'El valor actual del contador.'
    },
    {
      name: '1 (actions)',
      type: 'object',
      description: 'Un objeto con las acciones para manejar el contador.'
    }
  ]}
/>

### Acciones

<ApiTable
  data={[
    {
      name: 'inc',
      type: '(value?: number) => void',
      description: 'Incrementa el contador, por defecto en 1.'
    },
    {
      name: 'dec',
      type: '(value?: number) => void',

    {

### Parámetros

El hook `useCounter` acepta dos parámetros:
