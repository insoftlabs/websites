# useSwitch

@raddix/switch es un hook que proporciona accesibilidad y comportamiento para un componente switch.

<Demo>
  <Switch />
</Demo>

- Soporte completo para las funciones del navegador.

- Soporte de etiquetado para lectores de pantalla.
- Puede ser controlado o no controlado.
- API completamente tipificada

## Usage

Los siguientes ejemplos muestran cÃ³mo usar el componente Switch creado a continuaciÃ³n.
Importar todo o partes de <code>@raddix/switch</code>

```jsx
import useSwitch from '@raddix/switch';

const Switch = props => {
  const { switchProps, state } = useSwitch.Root(props);
  const { switchThumbProps } = useSwitch.Thumb(state);

  return (
    <button {...switchProps} className='switch'>
      <span {...switchThumbProps} className='switch-thumb'></span>
    </button>
  );
};
```

### Default Checked

Switches are not selected by default. To be checked by default
using <code>defaultChecked</code> prop

<Demo>
  <Switch defaultChecked />
</Demo>

### Read only

To control the state, use <code>checked</code> property for initial
value. The <code>onChecked</code> event is fired when the <code>checked</code> prop receives the new value.

<Demo>
  <Switch defaultChecked disabled />
</Demo>

### Label Form

Switch supports the integration with HTML tag Label

<Demo>
  <label htmlFor='dev' style={{ paddingRight: 20 }} id='dev-3'>
    Living Room Lights
  </label>
  <Switch defaultChecked id='dev' aria-labelledby='dev-3' />
</Demo>
