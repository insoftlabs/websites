# Installation

To use Raddix hooks, you can install the entire package or install the hooks individually.

## Global Installation

To install Raddix globally, you can use the following command:

```bash
npm install raddix
```

## Individual Installation

To install a Raddix hook individually, you can use the following command:

```bash
npm install @raddix/<hook-name>
```

For example if you want to install the `useToggle` hook, you can use the following command:

```bash
npm install @raddix/use-toggle
```
