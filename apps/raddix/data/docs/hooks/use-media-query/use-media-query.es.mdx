# useMediaQuery

El hook `useMediaQuery` se utiliza para determinar si el tamaño de pantalla actual coincide con una consulta de medios determinada.

## Instalación

```sh
npm i @raddix/use-media-query
```

### Ejemplo básico

También puede pasar un número como consulta; en este caso coincidirá con la consulta de medios `(min-width: ${number}px)`.

## API

### Parameters

    <ApiTable
          data={[
            {
              name: 'query',
              type: 'string | number',
              description:
                'La consulta de medios para coincidir. Puede ser cualquier consulta de medios válida, como `(max-width: 768px)` o un número.',
              required: 'Yes'
            },
          ]}
        />

### Return value

Devuelve "true" si la consulta de medios coincide y "false" si no.
