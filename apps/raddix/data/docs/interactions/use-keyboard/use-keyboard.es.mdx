# useKeyboard

Es un custom hook que maneja las interacciones del teclado.

## Features

- Coincidencia de valores de teclado.
- La propagación se detiene por defecto.
- Puedes cambiar el tipo de verificador.
- API completamente tipada.

## Installation

Instale el custom hook desde su terminal.

```bash
yarn add @raddix/use-keyboard
```
